# 📚 My Book Collection - Frontend (BookManage_React)

## 概要
React + Viteで作成した書籍管理アプリのフロントエンドです。  
LaravelバックエンドAPIと連携して書籍情報の取得・登録を行います。

## 特徴
- 入力フォームとAPI取得データをマージして書籍を登録
- 複数候補の書籍をカードUIで直感的に選択可能
- 画像がない書籍にはデフォルト画像を表示
- モバイル・PCレスポンシブ対応
- 状態管理とMUIを組み合わせた操作性の高いフォーム設計

## 使用技術
- React: 18.3.1
- ReactDOM: 18.3.1
- React Router DOM: 7.0.2
- Vite: 6.2.0
- Node.js: 24.12.0
- npm: 11.6.2
- MUI (Material-UI): 6.2.1
